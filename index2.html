Embeding Files


 css file:  (inside the head tag)

<link rel="stylesheet" type="text/css" href="name.css">

javascipt file:  (inside the <body> or <head>)

<script src="file_name.js"></script>



Responsive Design

To create a responsive page, add the line of code inside the head tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

when the view point is #px or smaller:

@media only screen and (max-width: #px)
{
/* your code here*/
}

When width < height:

@media only screen and (orientation: portrait)
{
/* your code here*/
}


Add Icons To Webpage

To use the Font Awesome Icons, add this line of code inside the head tag:


<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">
To use the Google Icons, add this line of code inside the head tag:

<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
